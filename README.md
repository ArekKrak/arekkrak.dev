This portfolio and showcase website was developed as part of my full-stack development journey and serves as a central hub for presenting projects, technical skills, and interactive web development work.  

## Overview

A responsive showcase website built with HTML, CSS, Bootstrap, JavaScript, jQuery, PHP, and GSAP animations. The website highlights selected projects, technical skills, and experience while combining interactive UI elements, animated effects, and responsive layouts.

The site also includes a functional PHP contact form integrated by PHPMailer and Brevo SMTP.

   **Status:** Actively maintained and expanded as new projects are completed.
